# PRD: 부모로 (Bumoro)

## 목표
임신·출산·육아 과정에서 외롭고 막막한 한국 부모에게, "먼저 그 길을 지나간 부모"가 곁에서 알려주는 듯한
따뜻한 정보 플랫폼. 정부지원금 자동 매칭으로 트래픽을 확보하고, founder voice 콘텐츠로 락인한다.

## 사용자
- 1차: 임신~초등 입학 전 자녀를 둔 28~38세 부모 (정보 검색 능동적, 모바일 90%, 새벽 사용)
- 2차: 임신 준비 중인 예비 부모

## 핵심 기능 (Phase 1 MVP)

### 1. 정부지원금 자동 매칭
- 출산예정일 + 거주지 입력 → 받을 수 있는 지원금 전체 목록
- 비회원도 3개 미리보기 가능, 전체는 가입 필요
- 마감일 알림 (회원)

### 2. 부모로의 편지 (Letters from Bumoro)
- 주 1회 발행, founder voice 1인칭 에세이 (800~1,500자)
- 구독 시 이메일 자동 발송 (Resend)
- SEO: 긴 체류시간으로 구글 상위 노출 목표

### 3. 월령별 가이드
- 임신 주차별·생후 84개월 SEO 콘텐츠
- 마지막 한 줄은 반드시 founder voice 톤으로

### 4. 회원가입 / 부모 프로필
- 출산일, 자녀 수, 거주지 기반 개인화
- Supabase Auth (이메일 + 소셜 로그인)

## MVP 제외 사항 (Phase 2+)
- 예방접종 캘린더 (Phase 2)
- 새벽 3시 마음카드 (Phase 2)
- 우리 아이 일기장 (Phase 2)
- 우리 동네 부모 지도 (Phase 2 후반)
- 진짜 부모 후기 / 쿠팡 파트너스 (Phase 3)
- 커뮤니티 기능 (Phase 3+)

## 디자인
- 모바일 우선 (90% 모바일 사용자)
- 따뜻함이 hero. Utility(매칭 입력폼)는 스크롤 아래
- 컬러: #FF6B4A (오렌지) + #FAF3E7 (베이지) + #FFE9E0 (포근)
- 폰트: 프리텐다드
- "5초 안에 따뜻한 곳이구나 느끼게"

## 6개월 KPI
- M3: 가입 100명, 인스타 200명
- M4: 가입 500, MAU 2,000
- M6: 가입 5,000, MAU 50,000
